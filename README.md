## ğŸ“– Core Topics

### Week 1-2: Foundational Data Structures
- **Arrays**: Dynamic arrays, sliding window, two pointers
- **Linked Lists**: Singly, doubly, circular linked lists
- **Stacks**: Array/linked list implementation, monotonic stacks
- **Queues**: Circular queues, deques, priority queues
- **Hash Tables**: Collision resolution, custom implementations

### Week 3: Trees (Part 1)
- **Binary Trees**: Traversals (inorder, preorder, postorder, level-order)
- **Binary Search Trees (BST)**: Insert, delete, search operations
- **AVL Trees**: Self-balancing, rotations
- **Red-Black Trees**: Properties and operations

### Week 4: Trees & Graphs
- **B-Trees**: Multi-way search trees
- **Graphs**: Representations (adjacency matrix/list)
- **Graph Algorithms**:
  - BFS (Breadth-First Search)
  - DFS (Depth-First Search)
  - Dijkstra's Algorithm
  - Bellman-Ford Algorithm
  - Floyd-Warshall Algorithm

### Week 5: Advanced Structures
- **Heaps**: Min-heap, max-heap implementations
- **Priority Queues**: Heap-based implementation
- **Segment Trees**: Range queries and updates
- **Fenwick Trees (BIT)**: Efficient prefix sums
- **Tries**: Prefix trees for string operations
- **Suffix Arrays**: Pattern matching

### Week 6: Algorithm Paradigms & Analysis
- **Divide & Conquer**: Merge sort, quick sort, binary search variations
- **Dynamic Programming**: Memoization, tabulation, state optimization
- **Greedy Algorithms**: Activity selection, Huffman coding
- **Backtracking**: N-Queens, Sudoku solver, permutations
- **Complexity Analysis**: Big-O, Big-Î˜, Big-Î©, Amortized Analysis
